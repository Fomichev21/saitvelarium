# VPN Bot

Telegram-бот для продажи VPN-подписок без сайта.

## Что умеет бот

- показывает тарифы и создает счет вида `AT=01`
- отправляет пользователя на оплату по фиксированной SBP-ссылке
- уведомляет админов о новом счете
- дает админам кнопки `Принять` и `Отклонить`
- после подтверждения продлевает подписку и выдает доступ
- поддерживает промокоды, баланс, бан, рассылку и рефералку
- умеет автоматически создавать клиента в 3x-ui и выдавать ссылку подписки

## Как работает оплата

1. Пользователь выбирает тариф в боте.
2. Бот создает счет и показывает кнопку оплаты.
3. Бот отправляет этот счет админам.
4. Админ вручную проверяет перевод и нажимает `Принять` или `Отклонить`.
5. После `Принять` бот активирует подписку и отправляет пользователю ссылку.

## Переменные окружения

- `BOT_TOKEN`
- `OWNER_ID`
- `SUPPORT_USERNAME`
- `MANUAL_PAYMENT_URL`
- `PAYMENT_INVOICE_PREFIX`
- `APP_SECRET_KEY`

Для 3x-ui:

- `THREEXUI_BASE_URL`
- `THREEXUI_USERNAME`
- `THREEXUI_PASSWORD`
- `THREEXUI_INBOUND_ID` или `THREEXUI_INBOUND_REMARK`
- `THREEXUI_SUB_BASE_URL`
- `THREEXUI_SUB_PATH`
- `THREEXUI_CLIENT_PREFIX`
- `THREEXUI_FLOW`
- `THREEXUI_LIMIT_IP`
- `THREEXUI_TOTAL_GB`

## Запуск

```powershell
.\.venv\Scripts\python.exe main.py
```

## Важно

- если админов в базе еще нет, бот попробует отправить счет на `OWNER_ID`
- если 3x-ui не настроен, проект использует локальную генерацию доступа через `wg`
- для автоматической выдачи именно ссылки из 3x-ui нужно заполнить переменные `THREEXUI_*`
